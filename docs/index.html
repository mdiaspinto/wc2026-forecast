How to read it. Each scoreline is the single most-likely exact result (the mode of a Dixon-Coles goal model fit to the sharp market). closing = captured ~45 min pre-kickoff (the number to act on). preview = current non-closing line, will move before kickoff. Verdict reflects market-fit confidence (BET / CAUTION / SKIP). Probabilities are model estimates, not guarantees.
